⬡ HiRO CRM
DMV Territory · Shared Team Database
One-time Firebase setup required
This CRM uses Firebase Realtime Database so Don, Jay, Damon, and Arifi all see the same leads in real time — for free. Takes about 5 minutes to set up once.
1
Go to console.firebase.google.com and click "Add project". Name it hiro-crm. Disable Google Analytics (not needed).
2
In the left sidebar click Build → Realtime Database. Click "Create Database". Choose United States. Start in test mode (allows all reads/writes for 30 days — enough to get started).
3
Go to Project Settings (gear icon top left) → Your apps → click the </> Web icon. Register the app. Copy the firebaseConfig object.
4
Open index.html in a text editor. Find the section labeled PASTE YOUR FIREBASE CONFIG HERE near the top. Replace the placeholder values with your real values. Save the file.
5
Host it free on Replit (replit.com → New repl → HTML → paste the file) or GitHub Pages (upload to a repo, enable Pages in Settings). Share the URL with the team. Done.
⚠ Until you paste your real Firebase config, the CRM will show this screen. The file works perfectly once configured — everything persists and syncs in real time across all devices.
